# Парсер вакансий ПлюсЗвено

Рабочий MVP-сборщик вакансий по списку промышленных и строительных специальностей.

## Что собирает

- `hh.ru` через публичный API.
- `vpoiskerabot.ru` через публичный JSON endpoint `/api/search`, если он доступен.
- `avito.ru` и `rabota.ru`:
  - по умолчанию фиксируются как источники для интеграции, без обхода защит;
  - с флагом `--allow-html-scraping` парсер пробует взять открытый `JobPosting` JSON-LD из HTML, если площадка его отдает.

## Запуск

```powershell
cd C:\Users\t8950\opo-service
py .\parser\vacancy_parser.py --limit-specialties 5 --sources hh vpoiskerabot avito rabota --hh-area 113
```

Если `py` не установлен, нужен нормальный Python 3.10+.

## Выходные файлы

Файлы пишутся в `parser/out`:

- `vacancies_YYYYMMDD_HHMMSS.jsonl` - нормализованные записи.
- `vacancies_review_YYYYMMDD_HHMMSS.csv` - таблица для представителя ПлюсЗвена.

## Статусы

- `primary_import` - первичная информация собрана автоматически.
- `source_pending` - источник требует официального API, партнерского экспорта или ручного подтверждения разрешенного способа сбора.

После импорта представитель ПлюсЗвена уточняет:

- фактическую ставку;
- график;
- проживание;
- дорогу;
- СИЗ;
- НДФЛ;
- тип оформления;
- договор;
- контакт и полномочия работодателя.
